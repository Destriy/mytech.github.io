____ ОПИСАНИЕ

В фотогалерее реализована функция открытия изображения в новом окне,
отформатированном под размер изображения с учетом разрешения экрана.
В случае невозможности определения разрешения, изображения выводятся
в окне размером 640х480 с возможностью прокрутки изображения.


____ ПОСТАВКА

readme.txt    - данный файл
galery.htm    - файл-образец, содержащий фотогалерею
01-1.jpg      - мини-фотография
01-2.jpg      - нормальная фотография (малый размер)
01-3.jpg      - нормальная фотография (средний размер)
01-4.jpg      - нормальная фотография (большой размер - больше разрешения экрана)


____ УСТАНОВКА

В файле galery.htm следует изменить следующие параметры:

В теле <body> заменить в строчках

<a href="#" onClick="newWin('01-2.jpg')"><img border="0" src="01-1.jpg" width="137" height="38"></a>


следующие параметры:
01-2.jpg       - на название файла, который будет показываться в окне
                 (большое изображение);
01-1.jpg       - на название файла, который будет показываться сначала
                 (маленькое изображение);
width="137"    - 137 на ширину Вашего изображения;
height="38"    - 38 на высоту Вашего изображения.

Количество таких строк зависит от количества фотографий в Вашей галерее.
